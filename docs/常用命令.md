# 虚拟环境

创建虚拟环境
python -m venv venv

激活虚拟环境
source venv/bin/activate (Mac)
venv\Scripts\activate (Windows)

退出虚拟环境
deactivate

# 依赖管理
查看所有可用稳定版
pip index versions fastapi

安装成功后怎么验证
pip show fastapi

安装指定版本依赖包
pip install fastapi==0.135.2

卸载已经安装依赖包
pip uninstall fastapi

导出所有已经安装的依赖
pip freeze > requirements.txt

安装项目所需的所有依赖
pip install -r requirements.txt
