# Sakila数据库项目 - 配置与部署

## 概述

Sakila项目采用现代化的配置管理和部署策略，支持从开发到生产环境的无缝迁移。项目通过环境变量、配置文件和多环境支持，确保应用在不同部署场景下的灵活性和安全性。

## 配置管理架构

### 1. 配置层级结构

```
环境变量 (.env)        # 最高优先级，敏感信息
    ↓
应用设置 (config/settings.py)  # 主要配置类
    ↓
具体配置 (config/database.py等)  # 组件配置
    ↓
运行时配置 (main.py等)     # 应用启动配置
```

### 2. 配置来源优先级

1. **环境变量**：最灵活，支持容器化部署
2. **.env文件**：开发环境便捷配置
3. **默认值**：代码中定义的合理默认值
4. **配置文件**：可选的额外配置文件

3. **默认值**: 提供合理的开发默认值
4. **验证**: 应用启动时验证必要配置

## 应用设置类 (config/settings.py)

### 1. 设置类设计

```python
import os
from pathlib import Path
from typing import Optional

from dotenv import load_dotenv
from pydantic_settings import BaseSettings

# 加载环境变量
env_path = Path(__file__).parent.parent / ".env"
load_dotenv(env_path)

class Settings(BaseSettings):
    """应用设置，从环境变量加载"""

    # 数据库配置
    database_url: str = os.getenv("DATABASE_URL", "postgresql://postgres:postgres@localhost:5432/sakila")
    database_url_async: str = os.getenv(
        "DATABASE_URL_ASYNC",
        "postgresql+asyncpg://postgres:postgres@localhost:5432/sakila"
    )

    # FastAPI配置
    debug: bool = os.getenv("DEBUG", "True").lower() == "true"
    host: str = os.getenv("HOST", "0.0.0.0")
    port: int = int(os.getenv("PORT", "8000"))

    # 安全配置
    secret_key: str = os.getenv("SECRET_KEY", "your-secret-key-here-change-in-production")

    # API配置
    api_prefix: str = "/api/v1"

    class Config:
        env_file = ".env"
        case_sensitive = False

settings = Settings()
```

### 2. 配置特性

- **类型安全**: Pydantic自动进行类型转换和验证
- **默认值**: 为所有字段提供合理的默认值
- **环境加载**: 自动从`.env`文件加载
- **单例模式**: 全局`settings`实例

## 数据库配置 (config/database.py)

### 1. 双引擎配置

```python
from sqlalchemy import create_engine
from sqlalchemy.ext.asyncio import AsyncSession, async_sessionmaker, create_async_engine
from sqlalchemy.orm import sessionmaker

from config.settings import settings
from models.base import Base

# 同步引擎（用于迁移和脚本）
engine = create_engine(
    settings.database_url,
    echo=settings.debug,          # 调试模式下输出SQL
    pool_pre_ping=True,           # 连接前检查
    pool_recycle=300,             # 连接回收时间（秒）
)

# 异步引擎（用于FastAPI端点）
async_engine = create_async_engine(
    settings.database_url_async,
    echo=settings.debug,
    pool_pre_ping=True,
    pool_recycle=300,
    future=True,
)

# 会话工厂
SessionLocal = sessionmaker(
    autocommit=False,
    autoflush=False,
    bind=engine,
)

AsyncSessionLocal = async_sessionmaker(
    async_engine,
    class_=AsyncSession,
    expire_on_commit=False,
)
```

### 2. 连接池配置

- **pool_pre_ping**: 防止连接过期
- **pool_recycle**: 定期回收连接（300秒）
- **echo**: 调试时输出SQL语句
- **future**: 使用SQLAlchemy 2.0 API

### 3. 会话依赖

```python
def get_db():
    """同步数据库会话依赖"""
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

async def get_async_db():
    """异步数据库会话依赖"""
    async with AsyncSessionLocal() as session:
        try:
            yield session
        finally:
            await session.close()
```

def main():

    parser = argparse.ArgumentParser(description="初始化Sakila PostgreSQL数据库")

    # 步骤2: 创建数据库

        create_database()

    # 步骤3: 创建表

    # 步骤4: 加载数据

### 2. 初始化步骤详解

#### 步骤1: 连接检查
```python
def check_postgres_connection():
    """检查PostgreSQL是否可用"""
    try:
        import psycopg2
        conn = psycopg2.connect(
            host="localhost",
            port=5432,
            user="postgres",
            password="postgres",
            database="postgres"
        )
        conn.close()
        logger.info("✅ PostgreSQL connection successful")
        return True
    except Exception as e:
        logger.error(f"❌ PostgreSQL connection failed: {e}")
        return False
```

#### 步骤2: 数据库创建
```python
def create_database():
    """创建sakila数据库"""
    conn = psycopg2.connect(...)
    conn.autocommit = True
    cursor = conn.cursor()

    # 检查数据库是否存在
    cursor.execute("SELECT 1 FROM pg_database WHERE datname = 'sakila'")
    exists = cursor.fetchone()

    if not exists:
        cursor.execute("CREATE DATABASE sakila")
        logger.info("✅ Created 'sakila' database")
    else:
        logger.info("✅ Database 'sakila' already exists")
```

## 应用启动配置 (main.py)

### 1. 生命周期管理

```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    """生命周期上下文管理器"""
    # 启动事件
    logger.info("Starting up FastAPI application...")

    # 开发模式下创建表
    if settings.debug:
        from models import Base
        logger.info("Creating database tables (development mode)...")
        async with async_engine.begin() as conn:
            await conn.run_sync(Base.metadata.create_all)

    yield

    # 关闭事件
    logger.info("Shutting down FastAPI application...")
    await async_engine.dispose()
    engine.dispose()
```
